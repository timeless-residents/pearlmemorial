# Soul Carrier 詳細ヒアリングフォーム / Detailed Hearing Form

Google Form 作成用テンプレート

---

## フォーム設定

- **タイトル**: Soul Carrier 詳細ヒアリングフォーム / Detailed Hearing Form
- **説明文**:
  ```
  【日本語】
  初回相談ありがとうございました。
  調査を開始するにあたり、詳細な情報をお聞かせください。
  分かる範囲で構いません。後から追加・修正も可能です。

  【English】
  Thank you for your initial consultation.
  Please provide detailed information to help us begin the research.
  Fill in what you know - you can add or update information later.
  ```

---

## Section 1: 依頼者情報 / Your Information

### Q1. お名前 / Your Name *
- Type: Short answer
- Required: Yes

### Q2. メールアドレス / Email Address *
- Type: Short answer
- Required: Yes
- Validation: Email

### Q3. 電話番号 / Phone Number
- Type: Short answer
- Required: No
- Hint: 国際電話の場合は国番号から / Include country code for international

### Q4. 故人との関係 / Your Relationship to the Deceased *
- Type: Dropdown
- Required: Yes
- Options:
  - 子 / Child
  - 孫 / Grandchild
  - 甥・姪 / Niece/Nephew
  - 配偶者 / Spouse
  - 兄弟姉妹 / Sibling
  - その他 / Other

### Q5. 「その他」の場合、具体的に / If "Other", please specify
- Type: Short answer
- Required: No

### Q6. 日本語レベル / Japanese Language Level
- Type: Multiple choice
- Options:
  - ネイティブ・流暢 / Native/Fluent
  - 日常会話可能 / Conversational
  - 挨拶程度 / Basic greetings only
  - なし / None

### Q7. 希望する連絡手段 / Preferred Contact Method
- Type: Checkboxes
- Options:
  - メール / Email
  - 電話 / Phone
  - LINE
  - Zoom/ビデオ通話 / Video call
  - WhatsApp

### Q8. 連絡可能な時間帯（ハワイ時間）/ Available Hours (Hawaii Time)
- Type: Checkboxes
- Options:

  - 午前 9:00-12:00 / Late Morning

  - 夕方以降 17:00+ / Evening

## フォーム送信後のメッセージ

```
【日本語】
ご回答ありがとうございました。
内容を確認し、3営業日以内にご連絡いたします。
追加の情報や写真がありましたら、info@pearlmemorial.org までお送りください。

【English】
Thank you for your response.
We will review the information and contact you within 3 business days.
If you have additional information or photos, please send them to info@pearlmemorial.org
```

## GAS連携設定

フォーム送信時に以下を自動実行：

1. **確認メール送信** - 依頼者に自動返信
2. **チーム通知** - Slack/メールで新規ケース通知
3. **ケースシート作成** - スプレッドシートに自動整理
4. **ケースID発行** - SC-2025-001 形式

詳細は `hearing-form-gas.js` を参照
