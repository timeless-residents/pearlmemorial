# トキノコエ — 刊行物コンセプト

## 概要

「トキノコエ」は、Pearl Memorial / Soul Carrier / Toki Storage の活動を定期的に記録・発信する刊行物です。デジタルと紙の両方で発行し、国会図書館をはじめとする象徴的な機関に寄贈することで、「存在の証明」を刊行物というかたちでも実現します。

---

## なぜ刊行物か

Toki Storage は「1000年残るクォーツガラス」で存在を証明するプロジェクトです。しかし、存在の証明にはもうひとつの古典的な方法があります——**本・刊行物として公共図書館に収蔵されること**です。

国会図書館に納本された刊行物は、半永久的に保存されます。これはトキストレージの理念——「誰かがここにいた」ことを未来に届ける——と完全に重なります。

私たちの活動記録、支援した家族の物語、境界線上に立つ人々の声を、定期刊行物として残す。それは単なる広報ではなく、**活動そのものの延長**です。

## ISSN 取得と納本制度

### ISSN（国際標準逐次刊行物番号）

定期刊行物として正式に認知されるため、ISSN の取得を目指します。

- **申請先**: 国立国会図書館 ISSN日本センター
- **要件**: 定期的に刊行される逐次刊行物であること
- **メリット**:
  - 国際的な刊行物データベースに登録される
  - 図書館での収蔵・検索が容易になる
  - 刊行物としての信頼性・正式性が付与される

### 納本制度の活用

国立国会図書館法に基づき、国内で発行された出版物は国会図書館への納入が義務付けられています（納本制度）。これを積極的に活用し、私たちの活動記録を公的アーカイブとして残します。

- **納入部数**: 1部（民間出版物の場合）
- **代償金**: 出版時の定価の5割以内＋送料相当が支払われる場合あり
- **手続き**: 発行後速やかに国会図書館に送付

## コンテンツ構成

各号は以下のセクションで構成します。

### 定常セクション

1. **巻頭言「境界線から」**
   - 代表（佐藤拓也・美菜）による短いエッセイ
   - その季節に感じたこと、出会った境界線について

2. **ソウルキャリア便り**
   - 遺骨帰郷の活動報告（個人情報は匿名化）
   - 訪問した墓地・寺院・地域の紹介
   - 家族の声（許可を得たもの）

3. **トキストレージ通信**
   - クォーツガラスの新しい取り組み
   - 奉納報告
   - 技術的な解説コラム

4. **バウンダリストの窓**
   - 境界線上に立つ人々のストーリー
   - システムの外と内をつなぐ実践の報告

5. **トキノコエの台所**
   - マウイと日本をつなぐ食のコラム
   - 季節のレシピや食材の話

6. **お知らせ・次号予告**
   - イベント情報、パートナーシップ報告
   - 次号の特集予告

### 特集テーマ（号ごとに変更）

- 春号：新しい始まり、帰郷の季節
- 夏号：レジリエンス、ラハイナの記憶
- 秋号：収穫と感謝、パトロン報告
- 冬号：鎮魂と希望、年間活動報告

## 制作ワークフロー

```
1. 企画会議（発行2ヶ月前）
   ├── 特集テーマ決定
   ├── 取材対象・寄稿依頼
   └── ページ割り

2. 原稿作成（発行1.5ヶ月前）
   ├── Markdown で原稿執筆
   ├── 写真・図版の準備
   └── 翻訳（英語併記部分）

3. 編集・校正（発行1ヶ月前）
   ├── 内容チェック・事実確認
   ├── デザイン・レイアウト
   └── 関係者による最終確認

4. 発行・配布（発行日）
   ├── PDF版の公開（Web・メール）
   ├── 印刷版の発送（寄贈先・パトロン）
   └── 国会図書館への納本手続き
```

## この取り組みが生むもの

1. **存在の二重証明**: クォーツガラス（物理）＋ 図書館収蔵（制度）で、活動の痕跡を未来に残す
2. **信頼性の獲得**: ISSN付き定期刊行物を発行する団体としての社会的信用
3. **物語の蓄積**: 毎号の記事が、境界線上の人々の声のアーカイブになる
4. **パトロンとの絆**: 印刷物という「手に取れるもの」がパトロンとの関係を深める
5. **地域とのつながり**: 訪問先の図書館に寄贈することで、各地域との継続的な関係が生まれる
