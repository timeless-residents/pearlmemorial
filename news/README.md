# トキコエニュースレター — 刊行物コンセプト

## 概要

「トキコエニュースレター」（英題: Tokikoe Newsletter）は、Pearl Memorial が発行する季刊の逐次刊行物である。遺骨帰郷支援（ソウルキャリア）、記憶の超長期保存（トキストレージ）、境界線上の人々の記録（バウンダリスト・ムーブメント）の三つの事業領域を横断的に取り上げ、活動報告・論考・記録文学・実用情報を掲載する。

本誌は発行と同時に国立国会図書館への納本を行い、各号を公的アーカイブに半永久的に収蔵することを前提に編集される。「誰かがここにいた」ことを未来に届けるという事業理念を、刊行物そのものによって実践する取り組みである。

---

## 発行の趣意

### なぜ刊行物か

トキストレージは合成石英（クォーツガラス）に情報を刻み、物理的に1000年以上の保存を実現する。しかし、存在の証明にはもうひとつの古典的な経路がある。**刊行物として公共図書館に収蔵されること**である。

国立国会図書館に納本された出版物は、制度的に半永久保存の対象となる。物理媒体（石英）と制度的媒体（納本）の二重化によって、私たちの活動記録と、私たちが出会った人々の物語は、異なる時間軸で未来に届けられる。

### 広報ではなく記録

本誌は商品やサービスの販売促進を目的としない。各号に掲載される事例報告、論考、寄稿は、ハワイ・日本間の越境的な活動の一次記録として位置づけられる。読者への情報提供であると同時に、後年の研究者・関係者が参照しうるアーカイブ資料としての品質を目指す。

## 編集方針

### 基本原則

1. **一次記録としての正確性**: 固有名詞・日付・場所・数値は可能な限り検証可能な形で記載する。匿名化する場合はその旨を明記する。
2. **当事者の声の尊重**: 取材対象者の発言は、許諾を得たうえで原文に忠実に掲載する。編集部による要約・意訳を行った場合はその旨を注記する。
3. **越境性の記録**: ハワイと日本、制度の内と外、デジタルと物理——複数の境界を横断する事象を、どちらか一方の視点に偏ることなく記述する。
4. **持続可能な編集体制**: 少人数での制作を前提とし、無理のない頻度・分量で継続することを最優先とする。

### 個人情報の取扱い

- 遺骨帰郷の事例報告では、依頼者の**書面による掲載許諾**を必須とする。
- 許諾が得られた場合も、原則として氏名をイニシャルまたは仮名とし、具体的な住所は都道府県・州レベルまでとする。
- 写真掲載は別途許諾を取得する。

### 著作権

- 本誌に掲載された記事の著作権は、原則として執筆者に帰属する。
- Pearl Memorial は、本誌の発行・頒布・アーカイブ収蔵・デジタル公開に必要な範囲で、非独占的な利用権を保持する。
- 転載を希望する場合は編集部に連絡のこと。

## 納本制度と ISSN

### 納本制度

国立国会図書館法第24条に基づき、国内で頒布を目的として相当部数作成された出版物は、国会図書館への納入が求められる（納本制度）。本誌はこの制度を積極的に活用する。

| 項目 | 内容 |
|------|------|
| 根拠法令 | 国立国会図書館法 第24条・第25条 |
| 納入義務者 | 発行者（Pearl Memorial / Universal Need株式会社） |
| 納入部数 | 1部（民間出版物） |
| 納入期限 | 発行日から速やかに |
| 送付先 | 国立国会図書館 収集書誌部 国内資料課 |
| 代償金 | 出版時定価の5割以内＋郵送料相当（非売品の場合は全額が対象） |

### ISSN（国際標準逐次刊行物番号）

| 項目 | 内容 |
|------|------|
| 申請先 | 国立国会図書館 ISSN日本センター |
| 申請費用 | 無料 |
| 申請時期 | 創刊号発行後、第2号発行までに申請予定 |
| 付与条件 | 継続的な刊行実績の見込みがあること |

ISSN取得後は、表紙および奥付にISSNを明示し、国際逐次刊行物データベース（ISSN Portal）への登録を行う。

## 執筆ガイドライン

### 文体

- 本文は「です・ます」調を基本とする。巻頭言・編集後記はこの限りでない。
- 事例報告は客観的な記述を基調とし、当事者の発言は「 」で明示する。
- 感傷的な修飾を避け、事実と行為の記述を中心に据える。

### 表記

- 漢字使用は常用漢字表に準拠する。固有名詞はこの限りでない。
- 数字は原則として算用数字を用いる。慣用表現（「ひとつ」「一方」等）は漢数字を許容する。
- 外国語の人名・地名は初出時にカタカナ表記と原綴を併記する（例：ラハイナ（Lahaina））。
- 年号は西暦を用いる。
- 「——」（ダッシュ）は全角ダッシュ2個連続を用いる。

### 分量の目安

- 巻頭言: 800〜1200字
- 特集記事: 2000〜4000字
- 各連載セクション: 800〜1500字
- 編集後記: 300〜500字

| 時期 | 工程 | 内容 |
|------|------|------|
| 発行2ヶ月前 | 企画 | 特集テーマ確定、取材計画、寄稿依頼 |
